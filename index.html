Island View Visor de Fotogrametría 3D

Notas importantes:
  • Asegúrate que el archivo (.glb/.gltf) en Google Drive esté público o "Cualquiera con el enlace puede ver". Si no está público el navegador no podrá descargarlo.
  • Google Drive a veces aplica restricciones de CORS o redirecciones que impiden cargar el archivo directamente. Si ves errores en consola (CORS o 403), considera usar un hosting alternativo (GitHub Pages, Google Cloud Storage, AWS S3 o un CDN) o servir el archivo desde un servidor propio.
  • Para que Quick Look (iOS) funcione bien, necesitas proporcionar un archivo .usdz y usar el atributo ios-src="ruta/al/tu_archivo.usdz" en el <model-viewer>. Ejemplo: ios-src="https://tudominio.com/model.usdz".
  • Si quieres soporte AR en Android con Scene Viewer / WebXR, revisa que el modelo sea razonablemente liviano y bien optimizado.
